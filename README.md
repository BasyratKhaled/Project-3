# Software House Agency Website

A premium, high-performance, and responsive website designed for a modern software house. This project serves as a digital storefront, showcasing the agency's core development services, technical expertise, successful case studies, and a streamlined client onboarding pipeline.


## ✨ Key Features

- **Dynamic Service Catalog:** Clear, interactive sections highlighting core offerings (e.g., Full-Stack Web Development, Cybersecurity Architecture, Cloud Solutions).
- **Portfolio & Case Studies:** A filterable grid showcase displaying past successful projects, tech stacks used, and client success metrics.
- **Interactive Consultation Form:** A custom-built client intake form designed to capture project scopes and technical requirements seamlessly.
- **Performance Optimized:** Clean semantic markup, minified assets, and lazy-loaded images ensuring blazing-fast load times.
- **Fully Responsive Architecture:** Crafted with a mobile-first approach, ensuring pixel-perfect layouts across mobile, tablet, and ultra-wide displays.

---

## 🛠️ Tech Stack

- **Frontend Core:** HTML5, CSS3, Modern JavaScript (ES6+)
- **Layout Architecture:** CSS Grid, Custom Media Queries
- **Version Control:** Git & GitHub
